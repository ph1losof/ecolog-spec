# Changelog

All notable changes to the Ecolog specification will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-15

### Initial Release

- Complete specification for .env file parsing
- UTF-8 encoding requirements
- Three quoting modes (unquoted, single-quoted, double-quoted)
- Multiline support via quotes and backslash continuation
- Comment syntax (line and inline)
- Security considerations
- Compliance requirements
